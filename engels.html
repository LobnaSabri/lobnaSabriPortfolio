  • Business Software Project

    1. Discription

      2. In the second year of my studies, I had to make a web application in group. We got a customer who had a vision and we started developing that vision using use cases, wireframes and mock-ups. Once we were on the same page as the customer, we could start building the backend and frontend (Sapgui5) using SAP.

    2. My input

      3. Just like last year, we did not divide all the tasks, we took the initiative ourselves step by step to work on the code.       4. From this project, I learned that communication is very important, both with the client and within your group. What is also necessary is to always check the requirements of the customer so that you do not forget to implement anything.

  • E-project

    1. Discription
